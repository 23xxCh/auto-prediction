# 转子不平衡故障维护手册

## 故障类型：转子不平衡（Rotor Imbalance）

### 症状识别

- 振动信号在旋转频率处出现明显峰值
- 径向振动幅值增大
- 振动相位可能不稳定
- 设备整体振动能量增加

### 诊断标准

根据传感器数据判断：
- 振动信号在旋转频率（Hz）的幅值超过正常值的 2.5 倍
- 振动峰值超过 3σ 阈值
- 振动信号呈现周期性特征

### 维护步骤

1. **停机准备**
   - 断开设备电源
   - 确保转子完全停止

2. **动平衡检测**
   - 使用振动分析仪测量原始振动
   - 记录振动幅值和相位
   - 计算不平衡量

3. **转子检查**
   - 检查转子表面有无附着物
   - 检查转子有无腐蚀或磨损
   - 检查转子与轴的连接是否牢固

4. **平衡校正**
   - 在轻点添加配重
   - 逐步调整直至振动降至允许范围
   - 记录配重位置和大小

5. **振动验证**
   - 重新启停设备测试振动水平
   - 确认振动值在设备允许范围内
   - 存档平衡校正记录

### 所需备件

- 配重片（根据计算结果准备多种规格）
- 平衡胶泥或配重块
- 轴承（如果磨损导致振动异常）
- 润滑油

### 预防措施

- 定期清理转子表面附着物
- 每次检修后进行动平衡测试
- 建立转子平衡档案，记录历史数据
- 监控振动趋势，提前发现不平衡苗头

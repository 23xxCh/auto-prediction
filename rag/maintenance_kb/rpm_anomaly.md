# 转速异常故障维护手册

## 故障类型：转速异常（RPM Anomaly）

### 症状识别

- 实际转速偏离设定值
- 转速波动幅度超过正常范围
- 转速信号出现跳变或丢失
- 设备运行声音异常

### 诊断标准

根据传感器数据判断：
- 转速偏差超过设定值的 ±10%
- 转速标准差超过正常运行值的 3 倍
- 转速信号出现突变（阶跃变化超过 20%）

### 维护步骤

1. **控制系统检查**
   - 检查变频器/驱动器参数设置
   - 检查控制信号（4-20mA 或 0-10V）是否稳定
   - 检查 PLC 或控制器的输入输出信号

2. **电机检查**
   - 检查电机三相供电是否平衡
   - 检查电机接线是否松动
   - 测量电机绝缘电阻

3. **负载检查**
   - 检查负载是否卡阻
   - 检查传动机构是否顺畅
   - 确认负载在额定范围内

4. **传感器检查**
   - 检查转速传感器（光电/磁电/霍尔）安装位置
   - 清洁传感器表面异物
   - 使用示波器检查传感器输出信号

5. **驱动器调试**
   - 按照设备手册重新校准驱动器
   - 调整 PID 参数（如适用）
   - 进行空载和负载测试

### 所需备件

- 转速传感器（光电传感器或霍尔传感器）
- 变频器/驱动器模块
- 电机轴承
- 传动皮带（如磨损）

### 预防措施

- 每月检查转速传感器清洁度
- 每季度检查驱动器参数
- 建立转速历史档案，监控趋势变化
- 避免设备频繁变速运行
